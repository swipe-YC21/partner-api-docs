## EInvoice Document Errors

| Error Code            | Description                                                                                                                                                                                                                                                                                     |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `PORTAL_ERROR_{code}` | Error from E invoice portal. The {code} is the error code received from the e-invoice portal. ex: **PORTAL_ERROR_2247** here 2247 is the error code we got from e-invoice portal, we can check all einvoice portal error code [here](https://einv-apisandbox.nic.in/api-error-codes-list.html). |
