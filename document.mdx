We provide a comprehensive API solution for managing essential business documents such as invoices, subscriptions, pro forma invoices, sales returns, and purchase returns. Our APIs are designed to streamline your document creation, management, and retrieval processes with ease and efficiency.

## What We Offer

1.  [Document Creation](https://developers.getswipe.in/api-reference/document-v2/create-a-document) : Easily generate professional invoices, pro forma invoices, sales returns, and purchase returns tailored to your business needs.

### Steps to create a document:

    Checkout the create document endpoint in the API playground, use your API key to test the endpoint.
        <Card
        title="Create Document"
        icon="play"
        horizontal
        href="https://developers.getswipe.in/api-reference/document-v2/create-a-document" >

    Try Creating document

        </Card>

    If your API key is valid, this api endpoint returns a 200 response with document details in a specified JSON format.

    ### You can also manually create the documents on [Swipe Portal](https://app.getswipe.in/create/invoice)

    <img
        className="block "

2.  [Fetch PDF](https://developers.getswipe.in/api-reference/document-v2/get-document-pdf) : Access pdfs of already created documents.

        ### Steps to get document PDFs:

        Checkout the create document endpoint in the API playground, For doc_hash_id use hash_id received from create document.
        <Card
        title="Fetch Document PDF"
        icon="play"
        horizontal
        href="https://developers.getswipe.in/api-reference/document-v2/create-a-document" >
